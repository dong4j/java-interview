### How to contribute 贡献规范

目前来讲, 此项目仍然有很多地方需要完善, 此项目的文档收集于网络, 再者由于个人能力有限, 有些知识点未理解到位或者理解错误, 希望有经验的大佬前来帮我共同完善本项目, 鞠躬.
